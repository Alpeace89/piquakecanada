Canada Earthquakes Dashboard

About this dashboard

This website provides access to earthquake data from the Earthquakes Canada database, allowing users to download earthquake records without the limitations of the standard search interface.

The dataset is refreshed automatically once per hour and contains the most recent earthquake records across Canada. The downloadable CSV file corresponds directly to the dataset used to generate the map.

The map below shows earthquakes recorded over the last 30 days. Symbol size scales with magnitude and symbol colour represents earthquake depth.

Created and maintained by Alexander Peace, McMaster University. peacea2@mcmaster.ca
